RT-style confirm dialog

A Refined Confirm Dialog

Supports custom text, title, and button class

Click me!

Available options

title

The title to be displayed on top of the modal.

text

The textual content placed inside the modal body that informs user for an input.

close

the callback function to be called when a decision is made.

open

the callback function to be called when the popup is shown.

confirmButton

The text to be displayed on the confirm button.

cancelButton

The text to be displayed on the cancel button.

buttonClass

The class name to be applied to both confirm and cancel button.

alert

Turn the confirm dialog to an alert popup.

overlay

When this is set to true, the plugin generates the dialog with background overlay behind it. (default value: true)

Usage

Standard confirm dialog

 $('#demo1').rtConfirm();

Standard alert dialog

 $('#demo2').rtConfirm({alert:true});

Standard confirm dialog with custom text

 $('#demo3').rtConfirm({
  title:'moron test', 
  text: 'Is your father a man?', 
  confirmButton:'Yes', 
  cancelButton: 'I am not sure?'});

Custom callback when leaving the dialog

 $('#demo4').rtConfirm({ close: function( answer ){
  if(answer) 
    this.$el.text('I have been violated.');
  }});
